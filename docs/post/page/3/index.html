R-Forge projects and the svn repository

R projects available from R-Forge are version controlled using subversion (also known as svn). You can check out any repository using the “Anonymous Subversion Access”, in terminal: svn checkout svn://svn.r-forge.r-project.org/svnroot/projectname/ Each project may contain many packages (each in a separate folder).
